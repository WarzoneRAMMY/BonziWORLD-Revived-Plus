BonziWORLD Revived+ — Release v1.8.3

Initial v1.8.3 release notes and notable changes.

v1.8.3 — Initial release

  • Explosion visuals: explosion now appears opposite the character and is properly centered over targets.
  • Sprite spin sync: bonzi sprite spin is now connected to the visual sprite position (200×160 alignment).
  • Removed the black & white theme and restored original desktop images from backups.
  • Removed the login app showcase elements from the homepage for a cleaner login experience.
  • Added auto-save of login name to localStorage and automatic restoration on page load.
  • Added a dedicated changelog page and linked the "See Updates" button to it.

Notes

Most frontend changes were applied to /build/www/js/script.min.js. Please restart the server after pulling these changes to apply them.

Testing

  1. Open the site and log in to verify basic functionality.
  2. Test explosions and sprite visuals, and verify the saved username loads on page reload.

Contact & more info

For questions or to report issues, open an issue in the repository or email bonzi.world@outlook.com.
