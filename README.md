# BonziWORLD Renewed+  

### Wiki Feature
A realtime wiki is available under `/wiki`.  Editors must login using credentials defined in `server/settings.json` (see `sessionSecret` and `auth` fields).  The wiki supports search, trending, author filtering, version history, and basic AI moderation (requires `OPENAI_API_KEY`). Official Open Source Repository
